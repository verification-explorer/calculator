Calculator Documentation
========================

Welcome to the Calculator documentation! This is a simple yet powerful calculator
application built in Python, featuring core mathematical operations, calculation
history tracking, and an interactive command-line interface.

Features
--------

* **Core Operations**: Addition, subtraction, multiplication, division, power,
  square root, and modulo operations
* **History Tracking**: Keep track of your calculations with timestamps
* **Interactive CLI**: User-friendly command-line interface for quick calculations
* **Comprehensive API**: Use the calculator programmatically in your Python code

Getting Started
---------------

Install the calculator in development mode:

.. code-block:: bash

   pip install -e ".[dev]"

Then run the interactive calculator:

.. code-block:: bash

   calc
